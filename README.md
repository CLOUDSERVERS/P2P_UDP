# P2P_UDP
使用Android+Node.js实现基于UDP协议的P2P打洞测试，移动端

## 简介
+ 使用移动端（Android）和服务端（Node.js）来模拟内网穿透，并进行端对端数据交互的测试。
+ 自定义了简单的交互协议
+ 使用了TCP+UDP混合的方式进行内网穿透
+ Just for Fun!~
